## 2. 功能清单

> **来源**：PRD V1.1 §1.2.4 方案推送 + §1.3.5 推送时间；S13 §3.2~§3.3

### 2.1 推送类型

| 推送类型 | 触发时机 | 通道 | 来源 |
|----------|----------|------|------|
| 方案提醒 | 每日早 8:00 | 微信服务通知 | PRD §1.3.5 |
| Day 7/14/21 主动回忆 | Day N 早 8:00 | 微信服务通知 + 气泡 | PRD §1.6.2 |
| 连续未打卡关心 | 连续 3 天未打卡次日 9:00 | 微信服务通知 | PRD §1.3.5 |
| 方案完成庆祝 | Day 21 完成当日 | 抱抱卡 | PRD §1.8.1 |

### 2.2 DND（勿扰时段）

- 用户可设置 DND 时段（如 22:00~08:00）
- DND 时段内推送抑制，结束后顺延补发（不堆积）
- DND 跨 22:00 节点时：当日 22:00 前已推则不补，22:00 后未推则次日 9:00 补

### 2.3 推送订阅管理

- 微信服务通知需要用户首次授权订阅
- 用户可在个人中心关闭/开启各类推送
