## 3. 合规红线实现

> **来源**：PRD V1.1 §3.2 合规红线（6 条）

| 红线编号 | 禁止词/表述 | 替代话术 |
|----------|------------|---------|
| C-1 | 医疗表述（治疗/治愈/医生/处方/病症） | "养护方向" / "生活方式建议" |
| C-2 | 效果承诺（会变白/会变小/会提升） | 不承诺，只说"建议" |
| C-3 | 前后对比判断（比之前好/进步了） | "你的感受很重要" |
| C-4 | 数字量化（BMI/体重/三围） | 不量化 |
| C-5 | 颜值/外貌评判（变漂亮/颜值高） | 不评判 |
| C-6 | 强推打卡（"必须坚持"） | "今天做了就很棒" |

## 4. 心理健康危机识别与升级

> **来源**：PRD V1.1 §3.5 心理健康危机识别与升级

### 4.1 危机分级

| 级别 | 触发条件 | 产品响应 |
|------|----------|----------|
| L-Crisis（立即危机） | 用户表达自杀意念/自伤行为 | 停止 AI 交互，展示危机响应卡 |
| L-Serious（严重情绪） | 持续抑郁/无助/绝望情绪且无明确触发事件 | 展示倾听话术 + 专业资源 |
| L-Medical（医疗急迫） | 急性心理症状（惊恐发作、解离等） | 建议立即就医，提供急诊指引 |
